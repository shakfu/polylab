News

2011-05-05 LuaJIT 2.0.0-beta7 has been released
This release has some major fixes and new features for the LuaJIT FFI library. This is also the first release of the ARM port of the LuaJIT interpreter for Android, iOS (iPhone/iPad) and others. The ARM port of the JIT compiler will follow soon.
Please check the changelog for more details.
2011-05-05 LuaJIT 1.1.7 has been released
This is a minor bug fix release for the LuaJIT 1.1 branch.
Please check the changelog for more details.
2011-01-21 LuaJIT Roadmap 2011
The LuaJIT Roadmap 2011 has been published. It contains news about the LuaJIT FFI library, the sponsored ARM port and the planned release schedule for this year.
